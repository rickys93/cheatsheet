CSS selectors and specificity

CSS is used to improve the appearance of a website. It is built upon the HTML framework.

Selectors

Selectors must follow the following format: (selector) { key:value; }. Multiple keys and values can be used within the same curly bracket, and each key/value pair must be seperated with a semicolon (;).

Tags

HTML tags are like keywords which defines that how web browser will format and display the content (eg. <h1> denotes a heading with the highest priority).

HTML tags can be selected in css by using the tag type as the selector. For example, changing the text color to red for all h1 elements would be as follows:

h1 {
text-color : red;
}

Classes

Any HTML element can be attibuted a class (eg. <h1 class="class-name">, where class-name is user defined). Multiple elements can be attributed to the same class, meaning that using the class as a css selector will alter all elements in that class.

HTML elements classes can be selected in a css by using .class-name. For example, changing the text size to 15px for all elements in the class "heading" would be as follows:

.heading {
text-size : 15px;
}

IDs

Any HTML element can be attibuted an ID (eg. <h1 id="id-name">, where id-name is user defined).

HTML elements classes can be selected in a css by using .class-name. For example, changing the background color to white for element with id="important-text" would be as follows:

#important-text {
background-color : white;
}

Specificity

Specificity refers to the priority each css selector is given in regards customising the element. If multiple changes of the same type are made to an html element, the selector with the highest priority is used.

The order in which css selectors are prioritised is ID, class, tags (from highest to lowest priority).

Priority Type Example
1 ID #important-text
2 Class .heading
3 Tag h1

!important

It is possible to overide the priority by using !important next to the custom css. For example, the following would overide an ID selector, even though it is only a tag selector

h1 {
text-color : red !important;
}
